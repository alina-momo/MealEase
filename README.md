# MealEase: A Food Convenience Website

MealEase is a polished course-project website for a Chattogram-based meal convenience business. It includes normal meal ordering, subscription plans, frozen foods, ready-to-cook ingredient packs, special offers, checkout, cash payment, delivery schedule, and customer delivery notes.

## Website name
**MealEase**: a short memorable name combining fresh leaf/ingredients with food bites.

## Pages included
- Home
- Menu
- Subscription Plans
- Ready-to-Cook & Frozen Foods
- Offers
- About / Hygiene
- Checkout

## Features
- Green and white aesthetic UI
- Custom SVG logo
- Responsive design
- Product/category selection
- Subscription plan builder: 7 days, 14 days, 1 month, 2 months
- Delivery time selection
- Shopping cart / selected package summary
- Delivery address and note fields
- Cash on delivery payment option
- Demo order confirmation modal

## How to run
Open `index.html` in a browser. For best results, run through a local server:

```bash
python -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```

## Notes
This is a front-end demo website for a course business plan. It does not process real payments or submit orders to a server.
